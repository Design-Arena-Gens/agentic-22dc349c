## Lemon Pie Legends

This project showcases the five most reliable and crowd-favorite lemon pie recipes. Each card highlights timings, yields, ingredient breakdowns, step-by-step methods, and a bakerâ€™s note for foolproof results.

### Tech Stack

- Next.js App Router with React Server Components
- Tailwind CSS for styling
- TypeScript for type safety

### Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the curated lemon pie collection.

### Production Build

```bash
npm run build
npm run start
```

### Deployment

Deploy directly to Vercel:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-22dc349c
```
